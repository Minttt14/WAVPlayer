# WAV音效檔播放器

## 主要功能

提供使用者載入並播放本地端的 WAV 格式音效檔案。具備完整的播放控制功能，包含「單次播放」、「重複播放」、「停止播放」、「結束程式」，程式內建關閉防呆機制，當使用者嘗試結束程式時會跳出確認視窗，避免誤觸導致播放中斷。

## 使用方法

1.點擊「瀏覽」按鈕，系統會開啟檔案選擇對話方塊。為避免載入錯誤的檔案格式，系統已預設過濾條件，僅會顯示 .wav 格式的檔案供選取。選取完成後，檔案的完整路徑會自動帶入「音效位置」的文字方塊中

2.成功載入音效路徑後，可使用下方的「播放按鈕」群組進行控制

- 播放一次： 點擊後，系統會載入該路徑的音效檔並完整播放一次
- 重複播放： 點擊後，音效會以無限循環的方式持續播放
- 停止播放： 點擊後，系統會立刻中斷並停止當前的音訊輸出

3.當需要離開系統時，點擊「結束程式」按鈕或直接點擊視窗右上角的關閉圖示，系統會彈出「關閉確認」的提示視窗，選擇「是」才能離開程式，可以避免誤觸
